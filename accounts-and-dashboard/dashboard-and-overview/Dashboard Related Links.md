# Related Links

**SUMMERVILLE CREDIT UNION · CONSOLIDATED MEMBER GUIDE · DASHBOARD MODULE**

Module: nFinia Digital Banking > Dashboard > Related Links Sidebar

*Sources: Summerville Reports Series A (36 docs) + Series B (25 docs) | Features: nFinia Documentation Features Spreadsheet*

## 01 PRODUCT SUMMARY

The Related Links sidebar is a persistent panel on the right side of the Dashboard that provides one-click access to the member's most frequently used self-service tasks. Rather than navigating through multiple menus to reach common actions, the member can jump directly from the Dashboard to functions like changing their password, downloading e-statements, managing alerts, or setting up direct deposit.

The Related Links panel adapts between Light and Dark themes — in Light Mode, links appear as blue text on a white card; in Dark Mode, links appear as bright blue/white text on a dark card with arrow indicators. The functionality is identical in both themes.

These links complement the top navigation and the More menu. While the top navigation provides access to major modules (Accounts, Move Money, More), and the More menu exposes all secondary features, the Related Links sidebar surfaces the specific tasks that members use most often — reducing clicks and improving the efficiency of routine self-service actions.

## 04 END-TO-END WORKFLOW

*Navigation: The Related Links sidebar is on the Dashboard right panel, below the Credit Score meter. No navigation required — it is visible immediately after login.*

### Step 1 — Locate the Related Links Sidebar

After logging in, the member scrolls down the right panel of the Dashboard (below the Quick Transfer widget, Upcoming Payments, and Credit Score meter) to find the Related Links card. The card displays a vertical list of link labels, each with a right-arrow indicator.

### Step 2 — Click a Link

The member clicks any link label. The system navigates to the corresponding feature page within nFinia. The destination page loads in the main content area, replacing the Dashboard view. The top navigation bar remains visible, allowing the member to return to the Dashboard at any time.

### Step 3 — Complete the Task

The member completes their task on the destination page (e.g., changes password, downloads a statement, adjusts alert settings). When done, they click **Dashboard** in the top navigation to return.

## 05 LIGHT MODE vs. DARK MODE

| Element | Light Mode | Dark Mode |
|---|---|---|
| Card Background | White with subtle border | Dark navy/charcoal |
| Link Text Colour | Blue (#0066CC or similar) | Bright blue/white |
| Arrow Indicators | Grey chevrons | White/light chevrons |
| Hover State | Underline or background highlight | Background highlight with brighter text |

Both modes display the same links in the same order with identical functionality.
