# Transaction Search

## Summary

Transaction Search allows members to search for specific transactions across their accounts using multiple filter options. The search page provides fields for transaction type, date range (From/To), description keywords, and amount range. Search results display matching transactions with dates, descriptions, amounts, and associated account information. This feature helps members locate specific transactions quickly without manually reviewing entire transaction histories.

## Key Use Cases

- Find a specific charge or deposit by description
- Research transactions within a specific date range
- Filter transactions by amount to identify large transfers
- Locate a transaction by keyword
- Verify transaction details across multiple accounts

## End-to-End Workflow

**Step 1: Open Account Overview and navigate to Account Details**

The member clicks "Accounts" in the top navigation bar to open the Account Overview page. All accounts are listed with their balances and action buttons. The member clicks "Account Details" on any account row to open the full transaction history for that account.

<figure><img src="/.gitbook/assets/accounts-overview-group.png" alt="" width="480"><figcaption></figcaption></figure>

**Step 2: Access Transaction Search from Account Details**

On the Account Details page, the member clicks "Transaction Search" in the breadcrumb navigation or sidebar link. The Transaction Search page loads, displaying filter fields at the top including a Type dropdown, Date range fields (From and To dates), a Description text field for keyword search, and an Amount range input. The member enters search criteria in one or more filter fields and submits the search. Matching transactions appear below the filter area with dates, descriptions, amounts, and associated account information.

<figure><img src="/.gitbook/assets/transaction-search.png" alt="" width="480"><figcaption></figcaption></figure>

**Step 3: Review search results and additional transactions**

The member scrolls down the Transaction Search results page to view additional matching transactions. The Promotional and Self Service sidebar appears on the right side of the page, offering additional resources and service options.
