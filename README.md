# Summerville Credit Union — nFinia Digital Banking Guide

Welcome to the nFinia Digital Banking documentation — a curated GitBook containing only the features tracked in the **Feature Documentation Tracking** sheet. This guide covers the core nFinia retail banking modules alongside all documented vendor and API integrations.

## How to Use This Guide

Documentation is organized by module, mirroring the nFinia app navigation. Each section contains the corresponding CSUM step-by-step guide and/or integration reference doc.

| Section                     | What You'll Find                                                                   |
| --------------------------- | ---------------------------------------------------------------------------------- |
| **Authentication & Login**  | Login, MFA (CSUM-01, CSUM-22), Trusted Devices / Cryptographic Auth (CSUM-25)     |
| **Dashboard**               | Dashboard Activities (CSUM-02), Recent Activities (CSUM-21)                       |
| **Accounts**                | Account Overview (CSUM-03, CSUM-04), Credit Score (CSUM-24), Profile (CSUM-23)    |
| **Cards**                   | Card Controls & Alerts — Velera Retail Credit Card Alerts & Controls               |
| **Move Money**              | Hub (CSUM-05), Transfers (CSUM-06–08), FedNow (CSUM-09), Wire (CSUM-12), Zelle (CSUM-11), Bill Pay (CSUM-14), Quick Pay (CSUM-13), Skip-A-Pay (CSUM-18), RDC        |
| **More Menu**               | Alerts (CSUM-16), Inbox (CSUM-17), Personal Info (CSUM-19), Settings (CSUM-20)   |
| **Vendor Integrations**     | Prisma, Direct/Web Connect, ZenDesk, FICS API, BDI SSO, Velera, MeridianLink, Plaid IAV |

## About This Platform

The **nFinia** platform from Tyfone powers this digital banking experience across web browsers, iOS, and Android — with a unified feature set across all channels.

> **Tip:** Use the sidebar navigation to browse by module, or use the search bar to find a specific feature by name or CSUM number.
