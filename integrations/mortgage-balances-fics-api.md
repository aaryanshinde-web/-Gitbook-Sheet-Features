# Mortgage Balances — FICS API

**Vendor:** Banking Core — Fiserv (FICS)\
**Status:** Documented\
**Module:** Accounts — Loan & Mortgage

## Overview

The FICS API integration pulls mortgage loan balance data from the Fiserv core banking system and surfaces it within the nFinia member's account dashboard. Members can view their current mortgage balance, payment history, and upcoming payment details without leaving the digital banking app.

## Key Features

- Real-time mortgage balance display via Fiserv FICS API
- Upcoming payment due dates and amounts
- Payment history accessible from the account detail screen
- Seamless integration with the Accounts module
