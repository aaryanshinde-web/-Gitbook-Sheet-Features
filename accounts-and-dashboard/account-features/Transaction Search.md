# Transaction Search

## Summary

Transaction Search allows members to search for specific transactions across their accounts using multiple filter options. The search page provides fields for transaction type, date range (From/To), description keywords, and amount range. Search results display matching transactions with dates, descriptions, amounts, and associated account information. This feature helps members locate specific transactions quickly without manually reviewing entire transaction histories.

## Key Use Cases

- Find a specific charge or deposit by description
- Research transactions within a specific date range
- Filter transactions by amount to identify large transfers
- Locate a transaction by keyword
- Verify transaction details across multiple accounts

## End-to-End Workflow

**Step 1: Access Transaction Search and set filter criteria**

The member is viewing the Account Details page. The member clicks "Transaction Search" in the breadcrumb navigation or sidebar. The Transaction Search page loads, displaying filter fields at the top including a Type dropdown, Date range fields (From and To dates), a Description text field for keyword search, and an Amount range input. The member enters search criteria in one or more filter fields and submits the search. Search results appear below the filter area, displaying matching transactions with dates, descriptions, amounts, and the accounts where each transaction occurred.
