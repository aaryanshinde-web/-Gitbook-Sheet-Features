# Transaction Search

## Summary

Transaction Search gives members the ability to locate specific transactions across account history using keyword, amount, date range, and transaction type filters — without downloading a statement or calling the credit union. For business members reconciling accounts, preparing documentation for audits, or investigating a disputed charge, the search capability turns a potentially time-consuming review into a targeted lookup that returns results in seconds.

## Key Use Cases

Business members use Transaction Search to locate a specific vendor payment by name or amount when reconciling the business checking account against an invoice ledger. Members preparing for an IRS audit or loan application use date range filters to extract transactions within a defined period and confirm that specific credits or debits appear correctly. Operations staff investigating a reported discrepancy use search to isolate the transaction in question and retrieve the full posting detail — description, amount, date, and reference number — without pulling the full account statement.

## Step-by-Step Guide

**Step 1 — Open Transaction Search from Account Details**

Navigate to Account Details for the relevant account and locate the **Search Transactions** field or filter icon above the transaction list. Enter a keyword — such as a payee name, check number, or partial description — to filter the transaction history in real time.

**Step 2 — Apply Date Range and Amount Filters**

Use the date range picker to restrict results to a specific period, and optionally enter a minimum or maximum amount to narrow the results further. Combined filters allow members to isolate a single transaction in a dense transaction history — for example, finding a $4,250 ACH credit posted in March — without scrolling through unrelated entries.

<figure><img src="../../.gitbook/assets/account-details-checking.png" alt="" width="536"><figcaption></figcaption></figure>

**Step 3 — Review and Act on Results**

Click any search result to open the full transaction detail view, which shows the posting date, description, amount, and running balance. From the detail view, members can initiate an Account Inquiry if the transaction requires investigation or flag it for follow-up with credit union staff.
