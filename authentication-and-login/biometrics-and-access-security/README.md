# Biometrics & Access Security

Biometrics & Access Security covers the pre-login convenience and biometric authentication options available to Summerville CU members on the nFinia mobile apps. These features let members sign in without typing a full password while still relying on a strong underlying credential for binding.

## What's in this section

* **Face ID — CDA Biometric Auth** — Face ID enrollment, binding, fallback behavior, and the Claimed Device Auth handshake used on iOS.
* **Sneak Peek** — Pre-login account balance glimpse — what it shows, how it is enabled, and what it deliberately does not reveal.
