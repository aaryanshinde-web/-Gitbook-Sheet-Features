# Release Notes: Tax Information Summary 

## Summary

The Joint Owners & Beneficiary Names update enhances the account details
interface by providing clear visibility into all individuals associated
with a membership. This feature consolidates primary account holders,
joint owners, and beneficiaries into a single, organized table,
improving readability and operational efficiency for the members.

### Platforms : Online Banking, Mobile Banking (iOS & Android)

### Fix Version: 10.49

### Supported Cores: Correlation 

## Key Benefits for Members

1.  > **Consolidated View:** All associated names—whether joint owners
    > or beneficiaries—are grouped under a clear "Account Holder
    > Information" section.

## Frequently Asked Questions (FAQs)

**Q: How are different association types from the Corelation core
handled?**

A: nFinia pulls all available association types from the core, but
maintains a clean UI by only displaying rows for roles that are active
on the account. By default, the interface prioritizes the Primary Owner,
Joint Owners, and Beneficiaries. Specialized roles—such as Trustees,
Authorized Signers, or Contingent Beneficiaries—are dynamically injected
as additional rows only when data exists, ensuring each is clearly
identified with an explicit role-based label.
