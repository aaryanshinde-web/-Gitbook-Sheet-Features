# End User Support Portal (ZenDesk)

**Vendor:** ZenDesk\
**Status:** Documented\
**Module:** Member Support

## Overview

The ZenDesk integration provides nFinia members with access to a self-service support portal. Members can submit tickets, browse the knowledge base, and interact with support agents — all accessible from within the nFinia digital banking experience.

## Key Features

- In-app support ticket creation
- Knowledge base / FAQ integration
- Live chat and email escalation
- Ticket status tracking for members

> **Documentation Note:** The ZenDesk support portal is a backend integration; member-facing access is embedded within the nFinia app's Help or Support section.
