# Loan Origination SSO — MeridianLink

**Vendor:** MeridianLink\
**Status:** Documented\
**Module:** Move Money — Loans

## Overview

The MeridianLink integration provides a Single Sign-On flow from nFinia into MeridianLink's loan origination platform. Members can initiate new loan applications from within the nFinia app and are seamlessly authenticated into the MeridianLink portal to complete the application process.

## Key Features

- SSO handoff from nFinia to MeridianLink loan origination
- Supports personal loans, auto loans, and credit card applications
- Pre-populated member data reduces application friction
- Returning members can check existing application status

## Navigation

**nFinia App:** Move Money → Apply for a Loan → (SSO to MeridianLink)
