# Dashboard

The Dashboard is the first screen every authenticated member sees after completing login and multi-factor authentication. It serves as the command centre of the nFinia digital banking experience — a single, consolidated view that presents the member's complete financial picture without requiring navigation to individual account pages.

## What's on the Dashboard

The Dashboard is composed of six distinct zones, each documented in its own page below:

| Zone | Location on Screen | Purpose |
|---|---|---|
| **Account Summary Tiles** | Centre column, top | All account balances (Checking, Savings, Loans, CDs, Lines of Credit) displayed as individual cards with masked account numbers and current/available balances. Clicking any account tile navigates to the **Accounts** module (Account Overview & Transaction History). |
| **Quick Transfer / Quick Pay** | Right panel or centre widget | A compact transfer form allowing the member to move funds between their own accounts without leaving the Dashboard. See [Quick Pay & Quick Transfer](Dashboard%20Quick%20Pay.md). |
| **Upcoming Payments** | Right panel | Displays scheduled and pending payments with due dates and amounts, giving the member an at-a-glance view of what's coming. See [Upcoming Payments](Dashboard%20Upcoming%20Payments.md). |
| **Credit Score Meter** | Right panel or below accounts | A visual gauge showing the member's current credit score range (Poor → Excellent) with a numeric score and "Read More" link. See [Credit Score Widget](Dashboard%20Credit%20Score%20Widget.md). |
| **Related Links Sidebar** | Right panel, bottom | Quick-access links to common self-service tasks: Change Password, Download E-Statements, Manage Alerts, Direct Deposit Form, etc. See [Related Links](Dashboard%20Related%20Links.md). |
| **Footer Quick Links** | Page footer | Persistent links to Security Policy, Branch Appointments, Locations, Rates, and Text Banking. See [Dashboard Quick Links](Dashboard%20Quick%20Links.md). |

## Navigation from the Dashboard

The top navigation bar is always visible and provides access to the platform's core modules:

- **Dashboard** — returns to this screen from anywhere
- **Accounts** — opens the full Account Overview with all account types and transaction history
- **Move Money** — opens the Move Money Hub with all transfer and payment options
- **More** — expands a panel with additional features (Alerts, Settings, Recent Activities, My Insights, Credit Score, etc.)

A **search bar** at the top of the navigation allows members to type any feature name (e.g., "Reports", "Zelle", "Bill Pay") and jump directly to it.

## Key Interactions

Clicking on an **account tile** on the Dashboard does not open account details inline — it navigates the member to the **Accounts** feature, specifically the Account Overview page (CSUM-03), where they can see the full account list, drill into any account's transaction history, and access account-level services.

The **Quick Transfer widget** on the Dashboard is a shortcut — it provides the same transfer functionality available in Move Money > Transfer to Own Account, but without leaving the Dashboard context.
