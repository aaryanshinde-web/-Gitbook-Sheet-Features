# Velera API Integration

Velera API Integration covers the credit-card servicing surface nFinia exposes through Velera (formerly PSCU) for Summerville Credit Union. These pages document member-facing flows only — underlying Velera API contracts, webhooks, and operational monitoring live in Tyfone’s internal integration docs.

## What's in this section

* **Velera — Credit Card Management** — The Velera-backed credit-card detail screen — balance, available credit, recent transactions, and controls.
* **Velera — Dispute a Transaction** — The member-initiated transaction dispute flow, including reason codes and evidence upload.
* **Velera — Credit Card eStatements & Rewards** — eStatement access, rewards balance and redemption, and promotional offers.
