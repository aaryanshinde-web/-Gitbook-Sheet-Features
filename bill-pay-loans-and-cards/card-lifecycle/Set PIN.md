# Set Card PIN

_Module: Banking › Cards › Card Details › Set PIN_

## Summary

The Set Card PIN feature allows members to create or update the Personal Identification Number (PIN) for their debit or credit card directly within nFinia Digital Banking, without needing to visit a branch or call member services. The PIN is required for ATM cash withdrawals and for in-store chip-and-PIN purchases at point-of-sale terminals. PIN changes are enforced immediately after saving and apply consistently across both physical card use and digital wallet transactions.

This is a fully self-service capability available around the clock via web or mobile. Members who forget their PIN or want to rotate it for security hygiene can complete the change in under a minute — eliminating a common call center inquiry and giving members direct control over their card security.

## Key Use Cases

| Use Case | Description |
| --- | --- |
| **Set PIN for a new card** | Create a PIN for a newly issued or replacement card before first ATM or chip-and-PIN use |
| **Forgot current PIN** | Reset the PIN immediately through digital banking without calling support or visiting a branch |
| **Routine PIN rotation** | Update the PIN periodically as part of personal security hygiene |
| **PIN compromised** | Replace the PIN without delay when there is a suspected exposure or unauthorized card use |

## Step-by-Step Guide

_Navigation: Banking › Cards › [select card row] › Set PIN_

### Step 1 — Open the Cards Dashboard

From the top navigation, click **Cards** to open the Cards dashboard, which lists all debit and credit cards linked to your account. Locate the card you want to set or change the PIN for. Click the **Set PIN** link displayed on that card row — or open Card Details and click **Set PIN** from the Manage Card section. Either entry point leads to the same Set Card PIN screen.

<figure><img src="../../.gitbook/assets/Cards_List.png" alt="Cards dashboard listing all user cards" data-size="original"><figcaption><p>Step 1: Open the Cards dashboard and click <strong>Set PIN</strong> on the card row.</p></figcaption></figure>

### Step 2 — Enter New PIN

The Set Card PIN screen displays your selected card and two input fields: **New PIN** and **Confirm New PIN**. Enter your chosen 4-digit numeric PIN in both fields. Choose a PIN that is memorable but not easily guessed — avoid combinations such as 1234, birth years, sequential digits, or repeating numbers, as these are common targets in card fraud.

<figure><img src="../../.gitbook/assets/Set_PIN_Form.png" alt="Set Card PIN — enter new PIN" data-size="original"><figcaption><p>Step 2: Enter and confirm the new 4-digit PIN.</p></figcaption></figure>

### Step 3 — Confirm and Save

Once both fields contain matching entries, the **Set New PIN** button becomes active. Click it to save your PIN change. The update is applied immediately and your card can be used at ATMs and chip-and-PIN terminals straight away without any waiting period.

<figure><img src="../../.gitbook/assets/Set_PIN_Filled.png" alt="Set Card PIN — ready to submit" data-size="original"><figcaption><p>Step 3: With both fields filled, click <strong>Set New PIN</strong> to save.</p></figcaption></figure>

> **Note:** For security, the PIN is never displayed in plain text once it has been saved. If you forget your PIN at any point, return to this screen and set a new one — there is no "recover PIN" option, only a reset.
