# FI-Facing Documentation

**Owner:** Product Marketing · **Review cadence:** quarterly · **Source spreadsheet:** `Marketing Doc Repository v2.xlsx`

- [nFinia Retail](nfinia-retail/README.md) — 15 items · Datasheets, pamphlets, success stories, and Harmoney admin console material.
- [Releases](releases/README.md) — 71 items · Release notes for every version (10.36 → 10.52). Both Retail and Business.
- [Add-On Products](add-ons/README.md) — 21 items · Penni AI, nFinia Business, CDA, New Account Opening — modules layered on top of nFinia.
- [Skip-A-Pay & Quick Pay](sapqp/README.md) — 18 items · Loan-side products. Core-specific (DNA / UltraData / Symitar / Corelation).
- [3rd-Party Integrations](third-party/README.md) — 58 items · Integration material for every partner: Array, BioCatch, Datavisor, Doconomy, Eltropy, Engageware, Ensenta, Fingoal, Larky, Nuuvia, Payfinia, Paze, Plaid, Pocketnest, SavvyMoney, Velera, Vertifi, Zoom.
