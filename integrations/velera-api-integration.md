# Velera API Integration

**Vendor:** Velera\
**Status:** Documented\
**Module:** Cards

## Overview

The Velera API integration enables new card management enhancements within the nFinia platform. This includes expanded card control capabilities, real-time transaction alerts, and updated card lifecycle operations powered by the Velera processing network.

## Key Features

- Card activation and deactivation via Velera API
- Real-time card transaction notifications
- Enhanced card controls (freeze, block, spend limits)
- Seamless integration with nFinia's card management UI

> **Documentation Note:** Refer to the `velera.md` reference document for detailed API specifications and integration architecture. Also see [Retail Credit Card Alerts & Controls](./velera-retail-cc-alerts-controls.md) for the member-facing card controls flow.
